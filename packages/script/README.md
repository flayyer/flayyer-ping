# @flayyer/ping-script

Monitor and pre-render Flayyer URLs - Browser script

## Usage

```html
<script async defer src="https://.../ping.min.js"></script>
<noscript>
  <img src="https://ping.flayyer.host/v1/ping.gif" alt=""/>
</noscript>
```
